# Idar Jewelry Management System

A premium, royal-themed management system for high-end jewelry businesses.

## Features
- **Diamond Inventory:** 4Cs tracking (Carat, Color, Clarity, Cut).
- **Gold Item Management:** Real-time pricing based on market rates.
- **Arabic/English Support:** Full multilingual support with RTL layout.
- **Sales Archive:** Secure records of all transactions.
- **Workshop Tracker:** Monitoring repair jobs from receipt to delivery.
- **Digital Receipts & QR:** Professional PDF generation and SKU QR codes.
- **Privacy Mode:** One-click revenue blurring for dashboard security.

## Tech Stack
- Vanilla HTML5 / CSS3 / JavaScript
- LocalStorage for persistence
- External JSON configuration

## How to use
Open `index.html` in any modern web browser.
